# 06 — Use Cases

Concrete applications of AI PM in practice — by workflow and by industry.

## What This Section Covers

Use cases show what AI PM looks like when it's actually applied. Each entry follows a consistent structure: the problem, the AI opportunity, the implementation approach, the results, and the risks.

Use cases are organised into two types:
- **Horizontal use cases** — cross-industry applications (prioritisation, onboarding, search, etc.)
- **Vertical use cases** — industry-specific applications (fintech, healthtech, SaaS, etc.)

## How to Contribute to This Section

Suitable contributions for `06_Use_Cases/`:
- New horizontal use case articles
- Industry-specific use cases (vertical)
- Expansions or corrections to existing use cases

Every use case must include: Problem Statement, AI Opportunity (honest about what AI can't do), Implementation Approach, Outcome Indicators, and Risks and Mitigations.

Please follow the format standard in [CONTRIBUTING.md](../CONTRIBUTING.md).

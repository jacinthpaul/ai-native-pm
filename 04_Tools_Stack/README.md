# 04 — Tools Stack

A curated, use-case-first guide to tools AI PMs use — organised by workflow stage, not vendor.

## What This Section Covers

The AI tools landscape moves fast and is full of noise. This section cuts through that with opinionated, use-case-first guidance. Every tool listed here has a documented PM use case — not just "it has AI features."

## Editorial Policy

- Tools are listed because they solve a real PM problem, not because they're popular or well-funded.
- All tool entries include a `last_evaluated` date. Check it — tools change rapidly.
- We are vendor-neutral. No affiliate relationships, no sponsored entries.
- If a tool's AI features have degraded or improved significantly, open an issue with the `correction` label.

## Suggested Reading Order

1. [AI Tools for PM Workflows: A Curated Overview](./tools-for-pm-workflows.md)
2. Use the [Tool Evaluation Card Template](./template-tool-evaluation-card.md) to evaluate tools for your specific context.

## How to Contribute to This Section

Suitable contributions for `04_Tools_Stack/`:
- New tool evaluation cards using the template
- Updates to existing tool entries (with updated `last_evaluated` date)
- New category overviews for workflow stages not yet covered

Please follow the format standard in [CONTRIBUTING.md](../CONTRIBUTING.md). Use `template-tool-evaluation-card.md` for individual tool entries.

📖 First time here? How to get these values
  1. Jira URL — your Atlassian site address. Open Jira in your browser and copy everything up to .atlassian.net, e.g. https://yourcompany.atlassian.net.
  2. Jira Email — the email address you use to log in to Jira.
  3. Jira API Token — not your password! Create one at id.atlassian.com → Security → API tokens: click Create API token, give it a name (e.g. "Planning Poker"), then copy & paste it here. It is stored only in this browser.
  4. Story Points Field — the Jira custom field where saved estimates go. Most Jira Cloud sites use customfield_10016 ("Story point estimate"). Leave the default first; if "Save to Jira" fails, find your field ID under Jira Settings → Issues → Custom fields → Story point estimate.

Use Test Connection after saving to verify your credentials. ✅

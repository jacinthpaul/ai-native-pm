# 08 — Templates

Reusable, copy-paste-ready PM templates adapted for the AI era.

## What This Section Covers

Every template here is designed to be picked up, filled in, and used immediately. Templates include usage instructions at the top — delete them when you're done filling in the document.

## Template Index

| Template | What It's For |
|----------|--------------|
| [template-ai-prd.md](./template-ai-prd.md) | Full PRD for a feature that includes AI behaviour |
| [template-agent-design-brief.md](./template-agent-design-brief.md) | Pre-scoping brief for any agentic feature |

## How to Use a Template

1. Click the template file you need.
2. Copy the raw content.
3. Create a new file in the appropriate location for your project.
4. Fill in every `[BRACKET]` with your content.
5. Delete the `<!-- HOW TO USE -->` instructions at the top.
6. Delete any sections that don't apply to your project.

## How to Contribute a Template

Suitable contributions for `08_Templates/`:
- New PM templates adapted for AI use cases (discovery templates, evaluation frameworks, etc.)
- Updates to existing templates based on real-world use

All template files must:
- Be prefixed with `template-`
- Include a `<!-- HOW TO USE -->` block at the top explaining the template
- Use `[BRACKETS]` for all placeholder content
- Include a frontmatter metadata block

Please follow the format standard in [CONTRIBUTING.md](../CONTRIBUTING.md).
